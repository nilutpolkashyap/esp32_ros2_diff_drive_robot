## Project Details

### Software Used
- Arduino IDE
- micro_ros_arduino library
- M5Core2 library
- ROS 2 Humble Hawksbill

### Hardware used 
- M5Stack AWS-M5Core2 (ESP32)
- ESP32 Dev Module 3.3V
- Robot Base (TortoiseBot)
